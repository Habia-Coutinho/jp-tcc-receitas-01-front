# 🍲 Site Delícias na Cozinha 

Projeto **Front-End** desenvolvido como **Trabalho de Conclusão de Curso (TCC)** do programa **Jovens Programadoras do Senac**.

A aplicação consiste em um site de receitas culinárias criado para praticar conceitos fundamentais de desenvolvimento web, como **estrutura de páginas, estilização e interatividade**.

## 📌 Sobre o Projeto

Este projeto foi desenvolvido como **trabalho final do curso Jovens Programadoras**, um programa educacional voltado à **formação de mulheres na área de tecnologia**.

O objetivo do projeto é criar uma **aplicação web que apresente receitas culinárias** em uma interface **simples, organizada e acessível**.

Durante o desenvolvimento foram aplicados conhecimentos de:

- Estruturação de páginas web
- Estilização com CSS
- Manipulação de elementos com JavaScript
- Organização de projetos front-end

---

## 🎯 Objetivo do Projeto

O projeto foi criado para demonstrar as habilidades desenvolvidas durante o curso, incluindo:

- Construção de interfaces web
- Organização de código front-end
- Aplicação de conceitos de programação
- Desenvolvimento de um projeto completo para portfólio
